"Bilimli Nesil" zehin merkezinde geçirilýän bäsleşiklere, online barlagnamalar gatnaşmak üçin:

1. Bäsleşigiň ýa-da olimpiadanyň meselelerini yuklap alyň we çözüň. Häzirki wagtda resminamalar kabul edilýän bäsleşikleriň sanawyny esasy bölümden tapyp bilersiňiz. Ýumuşlara jogaplary ýörite görnüşde girizip bolýar (sahypada meselelerde göçürip alýar) ýa-da el bilen ýazyp, soňra skanirläp ýa-da surata düşürip bolýar. Eseri gözden geçirýän ýa-da surata düşürýän bolsaňyz, mazmunyň hemmesiniň faýlda bardygyna we jogaplaryň manysyny bermek üçin faýlyň ýeterlik derejede çözgüdine göz ýetiriň.

2. Degişli anketany yuklapalyň we dolduryň.

Elektron poçta salgysyny görkezmegi ýatdan çykarmaň, sebäbi elektron baýraklar (Hormat haty we Diplomlary) oňa iberiler we yerine ýetirilen ýumuşlaryň jogaplary bilen şahsy hasabyň arasynda baglanyşyk dörediler (yerine ýetirilen işiň ýagdaýyny görüp, gatnaşmagyň deslapky şahadatnamasyny göçürip alyp bilersiňiz we jemlenenden soň) baýraklar (Hormat haty we Diplomlary) elektron poçta salgysyna iberiler we ondan yukläp alyp bilersiňiz.

3. Anketada gatnaşmagyň saýlanan görnüşini görkezmeli. Baýrak (Hormat haty we Diplomlary) gatnaşmagyň görnüşine baglydyr. Gatnaşmagyň elektron görnüşi bilen baýraklary (Hormat haty we Diplomlary) diňe elektron görnüşinde alarsyňyz.

4. Gatnaşmak üçin töleg sahypasynda hasaba alyş tölegini tölemegiň ähli bar bolan usullary sanalandyr, şeýle hem hasaba alyş töleginiň mukdaryny hasaplamak üçin kalkulýator bar. Gatnaşmagyň bahasy barada jikme-jik maglumat, mowzuk tapgyryna gatnaşmagyň şertlerinde bar.

5. Bäsleşik üçin materiallar bilen arhiw dörediň - anketa, ýerine ýetirilen ýumuş we hasaba alyş töleginiň tölenendigini tassyklamak. Bank hasabymyza tölänimizde, resminamanyň skanirlenen nusgasy tassyklama hökmünde goşulmalydyr. Web sahypamyz ýa-da elektron pul arkaly kartoçka bilen töläniňizde töleg usulyny aýratyn faýlda (web sahypamyz), senesi, wagty, mukdary we düşündirişi (bar bolsa) görkezmeli.

6. Arhiwiňizi şahsy hasabyňyz arkaly materiallar bilen ýükläň. (Şahsy hasap -> "Subýekt olimpiadalary we ýaryşlar" bölümi -> Bäsleşik üçin iş ýükläň, soňra faýly saýlaň we "Iber" düwmesine basyň). Goşmaça usul, arhiwleri materiallar bilen Bilimli-nesil@gmail.com adresine ibermekdir. 5 günüň içinde arhiw işlener we işiň ýagdaýy şahsy hasapda görkeziler (Şahsy hasap -> Işiň ýagdaýyny barlaň we gatnaşmak şahadatnamasyny göçürip alyň).
